# Trip_planner
This project is an AI-powered Trip Planner that generates personalized travel itineraries based on user input such as location, budget, duration, and preferences.  It leverages multi-agent collaboration (Crew architecture) and local LLM capabilities using Ollama API to provide intelligent, dynamic, and customizable travel plans

🎯 Key Idea

Instead of a simple static planner, this system uses multiple AI agents working together (Crew) to simulate real-world planning:

🧠 One agent researches destinations
📍 One agent optimizes routes
💰 One agent manages budget
🗓️ One agent builds itinerary

👉 Result: Smart, structured, and realistic trip plans

⚙️ Tech Stack
🤖 AI Engine: Crew-based multi-agent system
🧠 LLM: Ollama API (local model inference)
🎨 Frontend: (streamlit)
📊 APIs: Maps / Location services
🚀 Features
✨ AI-generated travel itinerary
📍 Location-based recommendations
💰 Budget-aware planning
🗓️ Day-wise schedule generation
⚡ Fast response using local LLM (via Ollama)
🔄 Multi-agent collaboration (Crew)
🔄 How It Works
User Input (Location, Days, Budget)
        ↓
Crew AI Agents Activated
        ↓
Destination Research + Route Planning + Budget Optimization
        ↓
LLM (Ollama) Processes Data
        ↓
Final Structured Travel Plan Generated
